# Pivot Workspace

This folder is the main workspace for the next phase of product thinking and execution.

## What Lives Here

- [current-state-assessment.md](./current-state-assessment.md)
  Current diagnosis of the logged-in product, its working pipeline, and its main bottlenecks.
- [source documents](./source%20documents)
  Canon and doctrine source texts that matter for the next round of work.

## Working Use

- Put new strategy, product, and restructuring documents here.
- Treat the current-state assessment as the baseline diagnosis unless we explicitly revise it.
- Treat the source documents as reference material, not as the product truth by default.
- When a new proposal depends on one of the source texts, cite the exact file.

## Current Ground Truth

The load-bearing product loop is:

`source -> seed -> operate -> evidence -> receipt`

That loop should stay stable while naming, metaphor, and surface design remain negotiable.
