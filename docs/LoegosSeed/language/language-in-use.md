# Language In Use

Date opened: April 6, 2026
Status: Living evidence file
Purpose: Capture verbatim Lœgos phrases as they appear in real conversation, with attribution, context, and a plain-English gloss. This file is a receipt of use, not a teaching document. Where source-lineage.md proves the concepts existed in the founding texts, this file proves the language is now spoken.

## How to read this file

Each entry records a moment when someone used a Lœgos construction without translating it back into English. The point is to show that the shorthand became faster and more precise than the long form. Outsiders should be able to follow the conversation through the gloss column. Insiders should recognize the phrases they have already started using.

Add to this file when:

- Someone uses a shape (△ □ œ 𒐛) as a noun, verb, or adjective in normal speech
- Someone uses a signal name (green, amber, red, neutral, attested) without a noun next to it
- Someone uses a trust level (L1, L2, L3) as a description rather than a metric
- Someone says "compile," "seal," "receipt," "operator sentence" naturally in coordination talk
- A new compression appears that did not exist before

Do not add invented examples. Only verbatim use.

## Compression ratios observed

These are not theoretical. They are observed.

| Long form | Lœgos form | English words | Lœgos words / chars | Ratio |
|---|---|---:|---:|---:|
| "we need evidence" | "□" | 3 | 1 char | ~85% shorter |
| "that claim is unsupported by sources in this box" | "that's red" | 9 | 2 | ~78% shorter |
| "this is done and proven by reality return" | "𒐛" | 8 | 1 char | ~95% shorter |
| "we found where both sides agree" | "œ" | 6 | 1 char | ~88% shorter |
| "the deadline has no supporting evidence and the system will flag it" | "Friday won't be red" (negated) | 11 | 4 | ~64% shorter |
| "this read is grounded with full provenance and survived operate" | "L3 trust" | 9 | 2 | ~78% shorter |

The compressions are not arbitrary. Each one carries a meaning the long form cannot carry in fewer words: the shape names the type, the color names the evidence state, the trust level names the provenance depth. English requires three sentences for what the language carries in one character.

## What this file proves

The language taught itself to its own creators through use. No tutorial was given. No glyph reference was opened. The shorthand became natural because it was shorter and more precise than the English alternative.

That is the strongest possible validation a language can receive. A spec can describe a language. A renderer can display it. But a language is only real when people reach for it without thinking. This file is the receipt that the moment has happened.

## Submission rules for new entries

- Verbatim phrases only. Do not paraphrase. Do not invent.
- Date the entry. Even an approximate date is better than none.
- Name the speaker if known. Anonymous is fine if not.
- Note the screen or context. "On the phone discussing the deadline" is enough.
- Write the gloss in one line of plain English so an outsider can follow.
- If the phrase is a new compression that did not exist before, mark it with a `*` next to the date.
- Do not edit other people's entries to make them sound more polished. The receipt is the actual phrase, not the cleaned-up version.
