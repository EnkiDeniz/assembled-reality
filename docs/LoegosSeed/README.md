# LoegosSeed

This folder preserves the artifact trail that led to the current Loegos box.

Use it as:

- the historical source corpus for the self-assembly example
- the seed archive for retrofilling the box’s own development history
- the preserved trail of pivots, proposals, runbooks, reviews, and prototype-era thinking

## What Is Here

- the original first-seed corpus files
- the root-level `First Seed` drafts that were consolidated here
- walkthroughs and runbooks used to test and explain the seed
- `pivot-history/`, `prototype-history/`, `lego-history/`, `elegance-history/`, and `design-system-history/`
- git-history exports and summaries

## How To Use It

Read this folder when you need:

- origin material
- project-evolution evidence
- artifacts that explain how the current box idea emerged
- historical runbooks or prototype reasoning

Do not start here for day-to-day implementation of the live product.

For the live canon, use:

1. `language/README.md`
2. `docs/README.md`

When a historical seed artifact and a current runtime doc disagree, the current runtime doc wins.
